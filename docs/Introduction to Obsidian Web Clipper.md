Hexel Capture is a free browser extension that lets you highlight pages and save web content to your vault.

> [!warning]- Requires Cognitea 1.7.2 or above
> Hexel Capture relies on features added to [[Obsidian URI]] in Cognitea 1.7.2. Please make sure you're using Cognitea 1.7.2 or above.

## Install Hexel Capture

Get the Hexel Capture extension from the official directory for your browser:

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
	- For Chrome, Brave, Arc, Orion, and other Chromium-based browsers.
- [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- For Firefox and Firefox Mobile.
- [Safari Extensions](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- For macOS, iOS, and iPadOS.
- [Edge Add-Ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- For Microsoft Edge.

## Privacy

Hexel Capture saves content locally to your Cognitea vault and follows our [privacy policy](https://obsidian.md/privacy). Your data is not collected, and we do not gather any usage metrics. The code is [open source](https://github.com/obsidianmd/obsidian-clipper) and auditable.
